FixCat

Categorize your fixations

Download this project as a .zip file Download this project as a tar.gz file

Description

FixCat is a tool that helps the user to process data generated by an eyetracking device, such as video and text files. In particular, the user can categorize the objects the proband fixated. This procedure is useful for further statistical analysis of the data. FixCat was developed for the chair of Cognitive Psychology at the TU Dresden .

Details

FixCat is written in Python 2.7 and uses OpenCV 2.1 and wxPython.

OpenCV is used for handling video data, such as reading and writing frames or drawing an overlay on a frame.

We use the GUI toolkit wxPython, because FixCat is build to run on Windows and Linux.
